# aoc-starter-kit

Skeleton code for setting up Advent of Code for a new year.

## USAGE

The examples below assume you have already authenticated to adventofcode.com.
See `AUTHENTICATION` below for more information.

    aoc-get 2020 1

Gets the input data for 2020 day 1.

    aoc-get 2020

Gets the input data for all 25 days of 2020.

    aoc-get

Gets the input data for all AoC years (2015 to present).

## AUTHENTICATION

1. Open a browser.
2. Open the browser's developer tools.
   - Chrome: Cmd-Opt-I, then click on the Network tab.
   - Firefox: Cmd-Opt-I, then click on the Network tab.
   - Safari: Cmd-Opt-I, then click on the Network tab.
3. Load <https://adventofcode.com/2015/auth/login> in your browser.
